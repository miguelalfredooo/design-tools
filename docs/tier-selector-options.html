Above Prompt Input

Compact button group right next to the "Run" button—feels like part of the action.

  • Hard to discover

  • Cramped at small widths
OPTION C
Dropdown in Runner Header

Select menu in the runner's header bar, right where stage selection might live.

Focus Prompt

✗ Cons

  • Easy to miss (header area)
  • Menu closes after selection
  • Less visible when selected
